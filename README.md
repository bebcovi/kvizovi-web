Report
------

Make sure you have [Grunt](http://gruntjs.com/) installed:

```sh
$ npm install -g grunt-cli
```

LaTeX documents are located in `doc`, you can compile them with:

```sh
$ grunt docs
```

You can also watch for changes with:

```sh
$ grunt docs:watch
```
